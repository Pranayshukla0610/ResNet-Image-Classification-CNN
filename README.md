# ResNet-Image-Classification-CNN
Overview

This project implements the Residual Network (ResNet) architecture for image classification tasks. ResNet introduces skip connections (residual connections) that help overcome the vanishing gradient problem, enabling the training of very deep neural networks with improved performance and stability.

Features
Implementation of ResNet architecture
Residual Learning with Skip Connections
Transfer Learning using pretrained ImageNet weights
Data preprocessing and augmentation
Model training, validation, and evaluation
Prediction on custom images

Data Augmentation

Results

ResNet effectively learns deep feature representations through residual connections, achieving high classification accuracy while mitigating degradation and vanishing gradient issues.
